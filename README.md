# Freelancer Portfolio - Nikita Akulinin

A modern, responsive multi-page portfolio website for freelancers, built with HTML, CSS, and JavaScript.

## 🚀 Features

- **Modern Design**: Clean, minimalist interface with modern trends
- **Fully Responsive**: Works perfectly on all devices (desktop, tablet, mobile)
- **Fast Loading**: Optimized code without heavy dependencies
- **SEO-friendly**: Proper HTML structure and meta tags
- **Interactive**: JavaScript animations and effects
- **Easy Customization**: Simple file structure for quick editing

## ⚡ Quick Start

1. Download all project files
2. Open `index.html` in your browser
3. Or run a local server:

```bash
# If you have Python installed
python -m http.server 8000

# Or with Node.js
npx serve .

# Open http://localhost:8000 in your browser
```

## 🎯 Customization

### Changing Content

1. **Personal Information**: Edit text in HTML files
2. **Photos**: Replace image URLs with your own
3. **Colors**: Change CSS variables in `:root`
4. **Fonts**: Update Google Fonts link

### Main Color Settings

```css
:root {
    --primary-color: #6366f1;    /* Primary color (purple) */
    --secondary-color: #06b6d4;  /* Secondary color (cyan) */
    --accent-color: #f59e0b;     /* Accent color (orange) */
}
```
